# Us Again

A local-first, privacy-focused app for capturing peak connection and utilizing it during conflict.

## Getting Started

Since this project was initialized in a restricted environment, you need to install the dependencies manually.

1.  Open your terminal.
2.  Navigate to the project directory:
    ```bash
    cd /Users/kylenmcclintock/.gemini/antigravity/scratch/us-again
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```
4.  Start the development server:
    ```bash
    npm run dev
    ```

## Project Structure

-   `src/components`: Reusable UI components.
-   `src/views`: Main application screens (Home, Harvest, Crisis, Pulse, Profile).
-   `src/types.ts`: TypeScript definitions.
-   `src/data.ts`: Default data and constants.
-   `src/App.tsx`: Main application entry point.
